# 🏠 HomeHeaven

A clean and responsive home furniture and household goods store website.  
Built with pure HTML and CSS.

---

## 📌 Description

**HomeHeaven** is a frontend-only real estate website designed to showcase properties, services, and contact information in a modern and welcoming layout. The project focuses on clean design, readable structure, and responsive behavior across all screen sizes.

---

## 🧠 Features

- 🏡 Property listings showcase
- 👥 About us / team presentation
- 📦 Products and services section
- 📬 Contact page with information form
- 📱 Fully responsive — mobile, tablet, and desktop
- 🎨 Clean and modern visual design
- ⚡ Lightweight — no dependencies, instant load

---

## 🛠 Technologies Used

| Layer   | Technology |
|---------|------------|
| Markup  | HTML5      |
| Styling | CSS3       |

## 🚀 Quick Start

### Clone the Repository
```bash
git clone https://github.com/babayaga0x/HomeHeaven.git
cd HomeHeaven
```

### Open in Browser
```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

Or serve locally:
```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve .
```

Then visit `http://localhost:8080`.

- **Hero section** — full-width banner with headline and call-to-action
